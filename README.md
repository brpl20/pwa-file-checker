# Descrição
Esse script simples serve para verificar:
1. A integralidade do nome das pastas;
2. Identificar pastas que estão ocupando muito espaço;
3. Copiar os Modelos da pasta Raíz (zMODELOS) para a Pasta Modelos (MODELOS);
